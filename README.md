
# Smart Health Assistant & Diagnostic Web Portal

An IoT-based health monitoring system using **Raspberry Pi 4** for real-time vitals tracking and an AI-integrated web portal for health assessments.

## Features
- **IoT Vitals Tracking:** Real-time monitoring of body temperature and pulse rate.
- **Web Dashboard:** Visualization of health data with personalized recommendations.
- **AI Skin Analysis:** Image upload feature for automated skin health assessments.

## Tech Stack
- **Hardware:** Raspberry Pi 4, MAX30102 (Pulse), DS18B20 (Temp).
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3, JavaScript
- **AI/ML:** TensorFlow/Keras (Skin Analysis logic)
